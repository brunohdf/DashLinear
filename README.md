# DashLinear
Exercises strategy to construct a linear dashboard with fragment to manage each Card and a custom view to keep layout base.

Base structure seems like:
- DashboardFragment
    - FisrtCardFragment
      - ViewModel
      - WhiteCard
        - SuccessState
        - ErrorState
        - LoadingState
  - SecondCardFragment
      - ViewModel
      - WhiteCard
        - SuccessState
        - ErrorState
        - LoadingState
